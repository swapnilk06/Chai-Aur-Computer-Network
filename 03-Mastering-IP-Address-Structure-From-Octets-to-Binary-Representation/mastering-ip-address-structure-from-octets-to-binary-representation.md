### How IP structured?

a.b.c.d
--> where a,b,c and d are integers dots se separated hote hai in an IP
- This notation called as --> `Dotted quad notation`

Example of IP

192.168.1.4 - non negative integer

ye separated part hai with dot that is `Octets`
e.g. 
- 192.168.1 ==> 1st 3 octets
- 4 ==> last octet

- **192.168.1.4** This is the - Decimal representation of IP

-  **192.168.1.4** --> Actually represent of IP in Binary format


 **192.168.1.4** => in binary format => 11000000.10101000.00000001.00000100

Use binary format understand ki IP ka weight kitna hai?

- 8 bits per octet
- 4 octets --> 8 * 4 bits -> 32 bits

`IPv4` ==> is 32 bits/4 bytes
